# Salonova - E-commerce Website

Salonova is a basic e-commerce web application developed using PHP, HTML, CSS, and JavaScript. The project demonstrates core concepts of web development including dynamic content handling, product display by category, and order placement functionality.

## 🚀 Features

- Dynamic product listing by categories
- Product detail view with images and description
- Add and manage products (Admin panel)
- Order placement system
- Basic frontend UI for user interaction
- Category-based navigation

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Server: XAMPP (Apache)  
- Database: MySQL (if used)
